What is UIMOON ?

uimoon is a toolbox that allows development MATLAB graphical user interfaces using a Object-Oriented design, like many other languages toolkits. It's inspired in a mixture of wxPython and Java syntax. Basically, is a wrapper for ordinary MATLAB controls.
